
### 1043 Is It a Binary Search Tree (25分)

* 1,建立二叉排序树
```
void insert(node* &root, int data) {
	if (root == NULL) {
		root = new node;
		root->data = data;
		root->lchild = root->rchild = NULL;
		return;
	}
	if (data < root->data)
		insert(root->lchild, data);
	else
		insert(root->rchild, data);
}

```
