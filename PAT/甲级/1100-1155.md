
### 1102 Invert a Binary Tree (25分)
* 题意：翻转一棵二叉树,输出：line1：层次遍历，line2：中序遍历
* 思路：后序遍历遍历交换左右子树
```
void postOrder(int root){
    if(root==-1)///一开始想不通为什么root是-1,
        return ;
    postOrder(nd[root].lchild);
    postOrder(nd[root].rchild);
    swap(nd[root].lchild,nd[root].rchild);
}
```
